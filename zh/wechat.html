微信公众号操作手册

公众号没有开源代码,但它的流量策略可以从三类官方材料反推:后台指标体系(平台测什么)、运营规范(平台罚什么)、公开课表态(平台想要什么)。本手册每条结论标注证据等级。

⚠️ 证据口径:与 X 手册不同,本页是对闭源系统的反推。四档标注:官方=白纸黑字的规则文档;理念=张小龙/微信公开课表态;指标反推=后台暴露的数据维度,测什么≈优化什么;第三方=运营圈观察,未经官方确认,当传闻看。所有具体倍数(如"完读率 1.3 倍加成")均属第三方,本页只采信方向、不采信数值。

反推方法

三面镜子拼出闭源算法的轮廓:

  • 后台指标 = 算法的食谱。平台把哪些数据摆给你看(完读率、常读用户、在看、传播层级),就等于告诉你它在测什么、优化什么。
  • 处罚清单 = 算法的负权重。《运营规范》和《推荐运营规范》逐条写明不推荐什么、封什么——这是比任何泄露都可靠的"负面特征表"。
  • 官方表态 = 目标函数。张小龙的"1:2:10"和"社交推荐最符合人性",直接给出了系统的长期方向。

分发架构:三层漏斗

公众号 = 订阅打底 + 社交链放大 + 机器推荐决定上限。"轻订阅、重推荐"是官方既定方向。

理念 张小龙 2019:"社交推荐是最符合人性的获取信息方式";2021 年写下 1 : 2 : 10(关注 : 朋友 : 机器)的目标比例。

官方 "在看"已进化为朋友♡独立入口,位置排在订阅内容之前,微信团队持续扩大测试。

实证 36 氪小号"36氪未来消费":推荐流量占比从 2023-08 的 24.3% 涨到 2025-03 的 45.9%;其大号仅 2.2%——账号越小,推荐占比越高。

✅ 一句话结论
公众号已经不是"订阅制媒体"。对小号,约一半流量来自推荐——你的真实竞争场是机器推荐池和朋友♡社交链,订阅数只是地基。

推荐资格:三个开关(零或一,不是权重)

官方 与 X 的连续权重体系最大的不同:公众号大量使用二元门槛。以下任何一个不满足,推荐流量直接为零:

开关要求说明
① 原创必须声明原创官方原文:"转载和分组群发的内容将不会被推荐"。原创不是加分项,是入场券
② 全量群发不分组发表分组群发 = 放弃推荐资格
③ 推荐许可未勾选"不允许被平台推荐"一旦勾选不可撤销,发布前必查

原创声明本身也有门槛(官方):公共内容(法规/新闻/榜单)、营销宣传、非独家授权的翻译整理都不得声明原创;违规声明会被取消标识并处罚账号。

行动指南 ① 每篇发布清单

  1. 三开关自查:声明原创 ✓ 全量群发(不分组)✓ 未勾选"不允许推荐" ✓ —— 错一个,推荐归零
  2. 标题克制:准确概括内容,不惊悚、不命令式、不留半截话。标题党在这里是封号风险,不是流量技巧
  3. 为完读率设计结构:首段给出全文价值承诺;小标题 + 图表 + 加粗结论分段;长文不等于流水账
  4. 文末自然引导"在看"(不带任何奖励):在看直接喂朋友♡推荐流。绝不做"转发朋友圈领 XX"——那是 30 天封号红线
  5. 不在正文里导流:二维码、"点此看完整版"、引流到第三方——规范 5.4 直接取消推荐资格
  6. 发布后不用守第一小时:推荐周期 3–7 天,数据慢慢看;把守帖的时间拿去写明天的内容

行动指南 ② 新号起步(与 X 完全相反)

  • 不需要冷启动期:推荐池对新号开放(36 氪小号实证 + 第三方"新号友好"观察一致)。第一篇就按最高质量标准写,直接博推荐
  • 前 10 篇全部原创 + 垂直:既是推荐资格,也是让系统的内容标签尽快收敛到你的领域
  • 跑官方体检:后台【账号成长】→【账号检测】,官方从基础合规/推荐运营规范/账号状态/流量反馈四个维度自查——这是公众号版的"源码",每月跑一次
  • 别买粉:刷量会被完读曲线识别(正常读者文末跳出,刷量开头跳出),且粉丝数不直接换推荐流量——小号推荐占比反而更高,买粉是负资产
  • 起量指标看推荐占比:后台阅读来源里"推荐"+"看一看"的占比持续上升 = 内容过线;长期为零 = 先查三开关,再查内容是否撞了不予推荐清单

行动指南 ③ 常读用户培养

  • 固定发布时间(如每天 18:30):常读的本质是打开习惯,习惯靠节律养成——这是公众号版的"边权充电"
  • 栏目化:固定的系列名/结构让读者形成预期("每周一拆一个源码"比随机选题更养常读)
  • 盯常读用户数的周变化而非粉丝数:常读涨 = 关系变厚;粉丝涨常读不涨 = 涨的是路人
  • 发布频率服务于质量:第三方称高频更新会冲淡单篇推荐;不论真假,公众号没有 X 那种"每日在场"的边权压力,每天一篇高质量 > 三篇灌水在这里更成立

行动指南 ④ 数据复盘节奏

  • 单篇:第 3 天和第 7 天各看一次(推荐周期长,首日数据不定生死)。看四个数:送达阅读率(标题)、完读率(内容)、在看率(传播)、推荐占比(出圈)
  • 账号:每月一次:常读用户趋势 + 账号检测体检 + 阅读来源结构变化
  • 诊断决策树:送达阅读率低 → 改标题/封面;完读率低 → 改结构/砍长度;在看率低 → 结尾缺乏共鸣点/号召;推荐占比为零 → 查三开关和违规清单

常见问题 FAQ

Q:引导读者点"在看"算诱导吗?

官方诱导条款打击的是带奖励/强制的"分享至朋友圈"。文末自然引导在看(无奖励、无强制)是行业常规,目前不在打击清单内。红线是"利益交换":凡是"点在看/转发领资料",都踩进诱导定义。

Q:AI 辅助写作会被限流吗?

官方打击的是"AIGC 虚假信息"和"AI 一键生成的假大空内容"——关键词是虚假和低质,不是 AI 本身。AI 辅助 + 人工事实核查 + 真实信息增量,不在打击范围。但纯洗稿式 AI 量产,撞的是低质和不实两条线。

Q:公众号还吃发布时间吗?

权重在下降:推荐分发与推送时刻解耦(周期 3–7 天)。但订阅打开和在看链式反应仍有晚高峰效应,固定时间对常读习惯也有价值——保留固定发布时间,但不必为它焦虑。

Q:转载别人的好文章有流量吗?

官方明文:转载内容不会被推荐。转载只能吃订阅和转发流量,对小号约等于零。想做内容策展,用"原创评述 + 引用"的形式,而不是白名单转载。

Q:为什么我的推荐流量是零?

按顺序排查:① 三开关(原创/全量/未关推荐)→ ② 账号检测体检 → ③ 内容是否撞不予推荐清单(导流/营销/标题党/时效失效)→ ④ 完读率是否过低(质量门槛)。前三项是资格问题,二元的;只有第四项才是"内容不够好"。
